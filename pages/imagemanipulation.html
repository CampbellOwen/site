Image Manipulation

Patterns and graphics made by manipulating images into something new

Inspiration

Followed my curiosity to test various image editing features across several applications. Created as a meditative and explorative practice.

Explanation

The goal for this project was to turn photos I'd taken into entirely new works through the use of various features in image editing software. I adjusted the curve, applied texture filters, used content-aware image healing and other features to alter the image until the image had become something altogether new.

Context

Created as a passion project using Snapseed and Procreate.
