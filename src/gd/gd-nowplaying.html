NowPlaying

e-Ink Spotify Display

  • Idle State

    When there is no song playing.

  • Logo Process

    The use of a grpahics technique called 'Dithering' and of Spotify music streaming are key characteristics of this project. I combined these elements to create a well-representative brand for the project.

Context

Visual design for Spotify e-ink display project by Owen Campbell. Assets created in Figma.

Explanation

Displays information about the current playing song in Spotify on a black, white, and red e-Ink display. Includes the song name, album name, album artwork, artist name, track position in album, and track popularity.

Inspiration

Influenced by 90s UI, especially in typeface selection of Chicago, Andale Mono and Geneva.
